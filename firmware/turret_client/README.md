# Прошивка турели — Station Mode

## Описание

Эта прошивка для XIAO ESP32S3 Sense подключается к WiFi точке доступа
ровера и предоставляет управление pan/tilt серво и MJPEG стрим с камеры.

## Особенности

- **WiFi Station Mode**: подключается к "RoverAP", статический IP 192.168.4.2
- **Auto Reconnect**: автоматическое переподключение при потере связи
- **MJPEG Stream**: HTTP стрим с камеры OV2640 на порт 81
- **UDP Control**: приём команд pan/tilt на порт 4210
- **Dual Core**: UDP на Core 0, серво на Core 1

## Пины

| Компонент | GPIO |
|-----------|------|
| Servo Pan | 4 (D3) |
| Servo Tilt | 2 (D2) |
| Camera | Встроенная OV2640 |

## WiFi

| Параметр | Значение |
|----------|----------|
| Target SSID | RoverAP |
| Target Password | rover12345 |
| Static IP | 192.168.4.2 |
| Gateway | 192.168.4.1 |

## Протокол

### Входящие команды (UDP :4210)
```
PAN:{};TILT:{}\n
```

- PAN: -90..90 → map(180,0) — **ИНВЕРТИРОВАН**
- TILT: -90..90 → map(0,180) — прямой

### HTTP endpoints (порт 81)

| Endpoint | Описание |
|----------|----------|
| /stream | MJPEG стрим |
| /capture | Одиночный JPEG |
| /status | JSON статус |

### Пример /status
```json
{"pan":90,"tilt":90,"heap":123456,"rssi":-45}
```

## Установка

1. Открыть в Arduino IDE
2. Выбрать плату: **XIAO ESP32S3** или **XIAO ESP32S3 Sense**
3. Установить библиотеку **ESP32Servo**
4. Убедиться что ESP32 Camera library включена
5. Upload

## Порядок запуска

1. Сначала включить ровер (AP должен быть доступен)
2. Затем включить турель (подключится к AP)
3. Подключить телефон к "RoverAP"

## Отладка

Serial Monitor (115200 baud) показывает:
- Статус WiFi подключения
- Входящие команды
- Ошибки камеры

## Камера

- Формат: JPEG
- Разрешение: QVGA (320x240)
- FPS: ~15-20
- Качество: 12 (0-63, меньше = лучше)
