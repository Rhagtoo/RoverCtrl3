# Прошивка ровера — AP Mode

## Описание

Эта прошивка для XIAO ESP32S3 (не Sense) создаёт WiFi точку доступа,
к которой подключаются турель и телефон.

## Особенности

- **WiFi AP Mode**: создаёт сеть "RoverAP" с фиксированным IP 192.168.4.1
- **Motor Watchdog**: автоматическая остановка если нет команд >500ms
- **PCNT Encoders**: аппаратный квадратурный подсчёт для точного RPM
- **UDP Protocol**: приём команд на порт 4210, телеметрия на 4211

## Протокол

### Входящие команды (UDP :4210)
```
SPD:{};STR:{};FWD:{};LASER:{}\n
```

- SPD: игнорируется (legacy)
- STR: -100..100 → map(40,140) → серво руля
- FWD: -100..100 → abs()=мощность, sign()=направление
- LASER: 0/1

### Телеметрия (UDP :4211)
```json
{"bat":100,"yaw":0.0,"spd":75,"pit":0.0,"rol":0.0,"rssi":-45,"rpmL":120.5,"rpmR":118.3}
```

## Установка

1. Открыть в Arduino IDE
2. Выбрать плату: **XIAO ESP32S3**
3. Установить библиотеку **ESP32Servo**
4. Upload

## Отладка

Serial Monitor (115200 baud):
- `status` — показать состояние
- `stop` — остановить моторы
- `help` — список команд
