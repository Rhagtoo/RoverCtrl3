# Прошивка ровера — AP Mode (v2.4)

## Описание

Прошивка для ESP32-S3, создаёт WiFi точку доступа,
к которой подключаются турель и телефон.

## Особенности

- **WiFi AP Mode** — сеть "RoverAP", фиксированный IP 192.168.4.1
- **Motor Watchdog** — автоматическая остановка если нет команд >500ms (мотор + лазер)
- **PCNT Encoders** — аппаратный квадратурный подсчёт, 440 CPR на выходном валу
- **UDP Protocol** — команды :4210, телеметрия :4211 (каждые 500ms)
- **2 передачи** — GEAR 1 (макс 50%) и GEAR 2 (макс 100%)
- **Телеметрия STR** — отправка текущего угла руля для одометрии Ackermann

## Пины

| Компонент | GPIO | Примечание |
|-----------|------|------------|
| Motor IN1 | 6 | Направление (HIGH = вперёд) |
| Motor IN2 | 7 | Направление (HIGH = назад) |
| PWM Left | 4 | ШИМ мотора |
| PWM Right | 5 | ШИМ мотора |
| Encoder L A | 12 | PCNT_UNIT_0 pulse |
| Encoder L B | 11 | PCNT_UNIT_0 ctrl |
| Encoder R A | 38 | PCNT_UNIT_1 pulse (⚠ может конфликтовать с PSRAM) |
| Encoder R B | 37 | PCNT_UNIT_1 ctrl (⚠ может конфликтовать с PSRAM) |
| Servo Steer | 13 | 50Hz, 500-2400μs |
| Laser | 17 | HIGH = вкл |

## Протокол

### Входящие команды (UDP :4210)

```
SPD:{val};STR:{val};FWD:{val};LASER:{val};GEAR:{val}\n
```

| Поле | Диапазон | Описание |
|------|----------|----------|
| SPD | любое | Игнорируется (legacy) |
| STR | -100..100 | Руль: map(-100..100 → 40°..140°) серво |
| FWD | -100..100 | Мотор: знак = направление, abs = мощность. Ограничен GEAR |
| LASER | 0 / 1 | Лазер вкл/выкл |
| GEAR | 1 / 2 | Передача: 1 = макс 50% FWD, 2 = макс 100% FWD |

### Телеметрия (UDP :4211, JSON, каждые 500ms)

```json
{"bat":100,"yaw":0.0,"spd":75,"str":42,"pit":0.0,"rol":0.0,"rssi":0,"rpmL":120.5,"rpmR":118.3}
```

| Поле | Тип | Описание |
|------|-----|----------|
| bat | int | Заглушка (всегда 100) |
| yaw | float | Заглушка (всегда 0.0) |
| spd | int | abs(FWD) — текущая мощность мотора (0..100) |
| str | int | Текущая команда руля (-100..100), для одометрии |
| pit, rol | float | Заглушки (0.0) |
| rssi | int | Всегда 0 (AP mode, нет базовой станции) |
| rpmL | float | RPM левого колеса (инвертирован в v2.4) |
| rpmR | float | RPM правого колеса |

### Энкодеры

- GM37-520-EN: 11 PPR × 4 edges × 10:1 = **440 CPR** на выходном валу
- Сэмпл каждые 200ms
- rpmL инвертирован (левый энкодер подключён в обратной полярности)

## Установка

1. Открыть `rover_ap.ino` в Arduino IDE
2. Выбрать плату: **ESP32S3 Dev Module** (или XIAO ESP32S3)
3. Установить библиотеку **ESP32Servo**
4. Upload

## Отладка

Serial Monitor (115200 baud). Доступные команды:

| Команда | Описание |
|---------|----------|
| `status` | RPM L/R, FWD, STR |
| `stop` | Остановить моторы |
